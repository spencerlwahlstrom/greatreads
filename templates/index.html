Books

Displays all Books and their attributes.
You can add a new Book, update an existing Book, and delete an existing Book.
Deleting an existing Book will remove it from the intersection tables authors_books and books_genres, as well as any associated reviews from the reviews tables.

Authors

Displays all Authors and their attributes. Also displays all Books by Authors with the use of the authors_books intersection table.
You can add a new Author, update an existing Author, delete an existing Author, and add an existing Author to a Book which updates the authors_books intersection table.

Reviews

Displays all Reviews and their attributes.
You can add a new Review, update an existing Review, and delete an existing Review.

Genres

Displays all Genres and their attributes. Also displays all Books by Genres with the use of the books_genres intersection table.
You can add a new Genre, update an existing Genre, delete an existing Genre, and add an existing Genre to a Book which updates the books_genres intersection table.
