Books

Displays all Books and their attributes.
You can add a new Book, or update/delete an existing Book.
Deleting an existing Book will remove it from the intersection tables authors_books and books_genres, as well as any associated reviews from the reviews table.

Authors

Displays all Authors and their attributes. Also displays all Authors by Books with the use of the authors_books intersection table.
You can add a new Author, or update/delete an existing Author.
Add a new Author to Book relationship, or update/delete an existing Author to Book relationship. This updates the authors_books intersection table.

Reviews

Displays all Reviews and their attributes.
Add a new Review, or update/delete an existing Review.

Genres

Displays all Genres and their attributes. Also displays all Books by Genres with the use of the books_genres intersection table.
Add a new Genre, or update/delete an existing Genre.
Add a new Genre to Book relationship, or update/delete an existing Genre to Book relationship. This updates the books_genres intersection table.
