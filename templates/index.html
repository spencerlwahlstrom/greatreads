Books

Displays all Books and their attributes.
You can add a new Book, update an existing Book, and delete an existing Book.
Deleting an existing Book will remove it from the intersection tables authors_books and books_genres, as well as any associated reviews from the reviews tables.

Authors

Displays all Authors and their attributes.
[WIP] Soon you will be able add a new Author, update an existing Author, and delete an existing Author.

Authors_Books

Displays all Authors by Books with the use of the authors_books intersection table.
As of now, you can update or delete an existing Author to Book relationship, which updates the authors_books intersection table.

Reviews

Displays all Reviews and their attributes.
[WIP] Soon you will be able to add a new Review, update an existing Review, and delete an existing Review.

Genres

Displays all Genres and their attributes. Also displays all Books by Genres with the use of the books_genres intersection table.
[WIP] Soon you will be able to add a new Genre, update an existing Genre, delete an existing Genre, and add an existing Genre to a Book which updates the books_genres intersection table.
